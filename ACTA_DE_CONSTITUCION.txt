DESCRIPCIÓN DEL PROYECTO
───────────────────────────────────────────────────────────────────────────────

NOMBRE: Mi Primera App
DESCRIPCIÓN: Aplicación Android que permite a los usuarios ingresar su nombre 
             y recibir un saludo personalizado.

OBJETIVO: Crear una aplicación funcional para Android que demuestre los 
          conceptos básicos de desarrollo de aplicaciones móviles, incluyendo
          interacción con usuario, gestión de eventos y manipulación de 
          componentes visuales.

PLATAFORMA: Android
SDK UTILIZADO: Android API 37 (compileSdk: 37)
VERSIÓN MÍNIMA: API 24
LENGUAJE: Kotlin

───────────────────────────────────────────────────────────────────────────────
DETALLES DE IMPLEMENTACIÓN
───────────────────────────────────────────────────────────────────────────────

FUNCIONALIDAD PRINCIPAL:
- El usuario ingresa su nombre en el campo EditText
- Al presionar el botón "Saludar", la aplicación:
  * Valida que el campo no esté vacío
  * Si es válido: Muestra un saludo personalizado "Hola [nombre]"
  * Si está vacío: Muestra el mensaje por defecto "Bienvenido a Android"

CORRECCIONES Y MEJORAS REALIZADAS:
- Actualización de compileSdk de 36 a 37 para cumplir con requisitos de 
  dependencias modernas
- Implementación de contraste de color adecuado en componentes de entrada
- Migración de literales de texto a recursos de string (best practices Android)
- Aplicación de LayoutFidelity checks y validaciones de accesibilidad
